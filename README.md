# Film-Rental-Analytics-Project-Overview
The project provides a comprehensive analysis of film rental data to uncover insights into store performance, customer behaviour, category profitability and revenue trends. It demonstrates an end-to-end analytics workflow, starting from SQL-based data preparation to Power BI dashboard creation and storytelling.
